# parkrun, by the numbers — the edited edition (9 Jun 2026)

An *Our World in Data*-style data book on parkrun: ~2,900 free weekly 5K events
across 21 countries since 2004. This is the **editor's cut** — the full 42-figure
draft read by an editor and tightened to its strongest **38 figures**, with four
newly commissioned analyses folded in to close the gaps.

**Read it:** open `index.html` (live via GitHub Pages). 38 figures across six
chapters, each with a plain-language "how this was made — data, maths & stats" box.

Built from public parkrun results (the complete Bushy history plus a worldwide
event-history census) + Open-Meteo weather. Figures show only aggregates, never
individuals. Independent — not affiliated with or endorsed by parkrun.
